# Delta Spec: 项目列表和导航栏移动端优化

> **Type:** MODIFIED
> **Module:** project
> **Date:** 2026-02-19
> **Change:** jsonchange-name-mobile-responsive-adaptation

## 概述

为项目列表页面和导航栏增加移动端响应式支持，优化项目卡片布局、侧边栏导航、顶部导航栏等元素，确保在移动设备上提供良好的导航和浏览体验。

---

## 场景

### Scenario 1: 移动端项目列表网格布局

```gherkin
Given 用户在移动设备（屏幕宽度 < 768px）上访问项目列表页面
  And 项目列表包含多个项目卡片
When 页面加载完成
Then 项目卡片以单列方式垂直排列
  And 每个卡片占据全屏宽度（减去左右边距）
  And 卡片内边距适配移动端（p-4）
  And 卡片之间的间距至少为 16px
  And 用户可以垂直滚动查看所有项目
```

### Scenario 2: 移动端项目卡片触摸优化

```gherkin
Given 用户在移动设备上查看项目列表
  And 每个项目卡片包含项目名称、描述、操作按钮
When 用户点击项目卡片
Then 卡片的点击区域至少为 44x44px
  And 卡片的 active 状态有明显的视觉反馈
  And 项目名称字体大小至少为 16px
  And 项目描述字体大小至少为 14px
  And 操作按钮（如编辑、删除）的点击区域至少为 44x44px
```

### Scenario 3: 移动端侧边栏导航折叠

```gherkin
Given 用户在移动设备上访问任意页面
  And 页面包含侧边栏导航
When 页面加载完成
Then 侧边栏默认隐藏（折叠状态）
  And 顶部导航栏显示汉堡菜单按钮（☰）
  And 汉堡菜单按钮的点击区域至少为 44x44px
  And 用户点击汉堡菜单按钮后，侧边栏从左侧滑入
  And 侧边栏占据屏幕宽度的 80%（最大 300px）
```

### Scenario 4: 移动端侧边栏导航交互

```gherkin
Given 用户在移动设备上打开侧边栏导航
  And 侧边栏包含项目列表、设置、退出登录等菜单项
When 用户查看侧边栏
Then 每个菜单项的点击区域至少为 44x44px
  And 菜单项字体大小至少为 16px
  And 菜单项图标大小至少为 20x20px
  And 用户点击菜单项后，侧边栏自动关闭
  And 用户点击侧边栏外部区域或遮罩层，侧边栏关闭
```

### Scenario 5: 移动端顶部导航栏布局

```gherkin
Given 用户在移动设备上访问任意页面
  And 页面包含顶部导航栏
When 页面加载完成
Then 顶部导航栏固定在屏幕顶部（fixed top-0）
  And 导航栏高度至少为 56px
  And 导航栏左侧显示汉堡菜单按钮
  And 导航栏中间显示当前页面标题或项目名称
  And 导航栏右侧显示用户头像或操作按钮
  And 所有按钮的点击区域至少为 44x44px
```

### Scenario 6: 平板端项目列表双列布局

```gherkin
Given 用户在平板设备（屏幕宽度 768px - 1024px）上访问项目列表页面
  And 项目列表包含多个项目卡片
When 页面加载完成
Then 项目卡片以双列网格方式排列
  And 每个卡片占据屏幕宽度的 50%（减去间距）
  And 卡片内边距和字体大小与桌面端一致
  And 用户可以垂直滚动查看所有项目
```

### Scenario 7: 移动端创建项目按钮

```gherkin
Given 用户在移动设备上访问项目列表页面
  And 页面包含"创建项目"按钮
When 用户查看页面
Then "创建项目"按钮固定在屏幕右下角（floating action button）
  And 按钮大小至少为 56x56px
  And 按钮显示"+"图标
  And 用户点击按钮后，打开创建项目对话框（全屏模式）
```

### Scenario 8: 移动端项目列表空状态

```gherkin
Given 用户在移动设备上访问项目列表页面
  And 用户尚未创建任何项目
When 页面加载完成
Then 显示空状态提示："暂无项目"
  And 提示文字和图标在移动端清晰可见
  And 提示文字字体大小至少为 16px
  And "创建项目"按钮在空状态下显眼可见
```

## 变更规格

### 项目列表移动端布局

| 屏幕宽度 | 网格列数 | 卡片宽度 | 卡片间距 |
|----------|----------|----------|----------|
| < 768px (移动端) | 1 列 | `w-full` | `gap-4` |
| 768px - 1024px (平板) | 2 列 | `w-1/2` | `gap-6` |
| > 1024px (桌面端) | 3 列 | `w-1/3` | `gap-6` |

### 项目卡片移动端样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 内边距 | `p-4` | `p-4` |
| 项目名称字体 | `text-lg` | `text-lg` (18px) |
| 项目描述字体 | `text-sm` | `text-sm` (14px) |
| 操作按钮大小 | `w-8 h-8` | `w-11 h-11` (44px) |
| 最小高度 | 无限制 | `min-h-[120px]` |

### 侧边栏导航移动端样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 显示模式 | 固定显示 | 抽屉式（默认隐藏） |
| 宽度 | `w-64` (256px) | `w-4/5 max-w-[300px]` |
| 菜单项高度 | `h-10` | `h-11` (44px) |
| 菜单项字体 | `text-sm` | `text-base` (16px) |
| 图标大小 | `w-5 h-5` | `w-5 h-5` (20px) |

### 顶部导航栏移动端样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 高度 | `h-16` (64px) | `h-14` (56px) |
| 汉堡菜单按钮 | 不显示 | `w-11 h-11` (44px) |
| 标题字体 | `text-xl` | `text-lg` (18px) |
| 用户头像大小 | `w-10 h-10` | `w-10 h-10` (40px) |

### 创建项目按钮移动端样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 位置 | 页面顶部 | 右下角固定（FAB） |
| 大小 | `h-10` | `w-14 h-14` (56px) |
| 图标 | 文字"创建项目" | "+" 图标 |
| 样式 | `button` | `rounded-full shadow-lg` |
