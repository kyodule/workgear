# Delta Spec: 看板移动端布局适配

> **Type:** MODIFIED
> **Module:** kanban
> **Date:** 2026-02-19
> **Change:** jsonchange-name-mobile-responsive-adaptation

## 概述

为看板页面增加移动端响应式支持，包括看板列布局、任务卡片样式、任务详情页全屏模式等，确保在手机和平板设备上提供良好的浏览和操作体验。

---

## 场景

### Scenario 1: 移动端看板列垂直布局

```gherkin
Given 用户在移动设备（屏幕宽度 < 768px）上访问看板页面
  And 看板包含 3 个列（待办、进行中、已完成）
When 页面加载完成
Then 看板列以垂直方式堆叠显示（单列布局）
  And 每个列占据全屏宽度
  And 用户可以垂直滚动查看所有列
  And 列标题和任务卡片在移动端保持可读性
```

### Scenario 2: 移动端任务卡片触摸优化

```gherkin
Given 用户在移动设备上查看看板
  And 看板列中包含多个任务卡片
When 用户点击任务卡片
Then 卡片的点击区域至少为 44x44px
  And 卡片内边距增加，便于触摸操作
  And 卡片文字大小适配移动端（不小于 14px）
  And 卡片的 hover 效果替换为 active 效果
```

### Scenario 3: 移动端任务详情全屏模式

```gherkin
Given 用户在移动设备上点击任务卡片
When 任务详情对话框打开
Then 对话框以全屏模式显示（占据整个视口）
  And 顶部显示返回按钮和任务标题
  And 标签页（Timeline、Flow、Artifacts）切换为底部固定导航
  And 内容区域占据剩余空间，可垂直滚动
  And 用户点击返回按钮或向下滑动可关闭详情页
```

### Scenario 4: 移动端创建任务对话框适配

```gherkin
Given 用户在移动设备上点击"创建任务"按钮
When 创建任务对话框打开
Then 对话框以全屏或底部抽屉模式显示
  And 表单输入框高度增加到至少 44px
  And 输入框字体大小至少为 16px（避免 iOS 自动缩放）
  And 按钮高度增加到至少 44px，便于触摸
  And 用户可以通过向下滑动或点击取消按钮关闭对话框
```

### Scenario 5: 移动端看板列拖拽禁用

```gherkin
Given 用户在移动设备上访问看板页面
  And 桌面端支持拖拽任务卡片在列之间移动
When 用户尝试拖拽任务卡片
Then 拖拽功能在移动端自动禁用
  And 用户可以通过点击卡片进入详情页，然后修改状态来移动任务
  And 或者提供移动端专用的"移动到"按钮
```

### Scenario 6: 平板端看板列双列布局

```gherkin
Given 用户在平板设备（屏幕宽度 768px - 1024px）上访问看板页面
  And 看板包含 3 个列
When 页面加载完成
Then 看板列以双列布局显示（每行 2 个列）
  And 第三个列换行显示在第二行
  And 列宽度自适应平板屏幕
  And 用户可以垂直滚动查看所有列
```

### Scenario 7: 移动端空状态提示适配

```gherkin
Given 用户在移动设备上访问看板页面
  And 看板列为空，没有任务卡片
When 页面加载完成
Then 显示空状态提示："暂无任务"
  And 提示文字和图标在移动端清晰可见
  And "创建任务"按钮在移动端易于点击（至少 44x44px）
```

### Scenario 8: 移动端看板列标题和操作按钮

```gherkin
Given 用户在移动设备上查看看板列
  And 列标题包含列名称和任务数量
When 用户查看列标题
Then 列标题字体大小适配移动端（至少 16px）
  And 列标题右侧的操作按钮（如添加任务）点击区域至少为 44x44px
  And 列标题在移动端保持固定，滚动时不会被遮挡
```

## 变更规格

### 看板列布局响应式断点

| 屏幕宽度 | 布局模式 | 列宽度 | 说明 |
|----------|----------|--------|------|
| < 768px (移动端) | 单列垂直堆叠 | 100% | 每个列占据全屏宽度 |
| 768px - 1024px (平板) | 双列网格 | 50% | 每行显示 2 个列 |
| > 1024px (桌面端) | 水平滚动 | 固定宽度（如 320px） | 保持现有布局 |

### 任务卡片移动端样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 内边距 | `p-3` | `p-4` |
| 字体大小 | `text-sm` | `text-base` |
| 最小点击区域 | 无限制 | `min-h-[44px]` |
| hover 效果 | `hover:bg-muted/50` | `active:bg-muted/50` |
| 拖拽功能 | 启用 | 禁用 |

### 任务详情对话框移动端模式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 显示模式 | 居中对话框 | 全屏 |
| 宽度 | 固定宽度（如 800px） | `w-full` |
| 高度 | 自适应内容 | `h-full` |
| 标签页位置 | 顶部 | 底部固定导航 |
| 关闭方式 | 点击遮罩或关闭按钮 | 返回按钮或向下滑动 |
