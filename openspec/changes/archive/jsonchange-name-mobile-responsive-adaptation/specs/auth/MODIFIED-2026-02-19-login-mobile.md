# Delta Spec: 登录页面移动端适配

> **Type:** MODIFIED
> **Module:** auth
> **Date:** 2026-02-19
> **Change:** jsonchange-name-mobile-responsive-adaptation

## 概述

为登录页面增加移动端响应式支持，优化登录表单、按钮、错误提示等元素的显示和交互，确保在移动设备上提供良好的登录体验。

---

## 场景

### Scenario 1: 移动端登录页面布局

```gherkin
Given 用户在移动设备（屏幕宽度 < 768px）上访问登录页面
When 页面加载完成
Then 登录表单占据屏幕中央
  And 表单宽度为屏幕宽度的 90%（最大 400px）
  And 表单内边距适配移动端（p-6）
  And Logo 和标题在移动端清晰可见
  And 表单背景和边框在移动端保持可见
```

### Scenario 2: 移动端登录表单输入框优化

```gherkin
Given 用户在移动设备上查看登录表单
  And 表单包含用户名和密码输入框
When 用户点击输入框
Then 输入框高度至少为 44px
  And 输入框字体大小至少为 16px（避免 iOS 自动缩放）
  And 输入框获得焦点时，页面自动滚动到输入框位置
  And 虚拟键盘弹出时，表单自动调整位置
  And 输入框的 label 在移动端清晰可见（字体至少 14px）
```

### Scenario 3: 移动端登录按钮触摸优化

```gherkin
Given 用户在移动设备上填写完登录表单
When 用户点击"登录"按钮
Then 按钮高度至少为 44px
  And 按钮宽度占据表单全宽
  And 按钮字体大小至少为 16px
  And 按钮的 active 状态有明显的视觉反馈
  And 按钮在加载状态下显示 loading 图标
```

### Scenario 4: 移动端错误提示显示

```gherkin
Given 用户在移动设备上提交登录表单
  And 登录失败（如用户名或密码错误）
When 错误提示显示
Then 错误提示在移动端清晰可见
  And 错误提示字体大小至少为 14px
  And 错误提示颜色在移动端清晰可辨（支持 Dark Mode）
  And 错误提示不会被虚拟键盘遮挡
```

### Scenario 5: 移动端忘记密码链接

```gherkin
Given 登录表单包含"忘记密码"链接
  And 用户在移动设备上查看登录页面
When 用户查看表单底部
Then "忘记密码"链接的点击区域至少为 44x44px
  And 链接字体大小至少为 14px
  And 链接颜色在移动端清晰可见
  And 链接与登录按钮之间的间距至少为 16px
```

### Scenario 6: 平板端登录页面布局

```gherkin
Given 用户在平板设备（屏幕宽度 768px - 1024px）上访问登录页面
When 页面加载完成
Then 登录表单宽度为 500px（居中显示）
  And 表单内边距和字体大小与桌面端一致
  And 输入框和按钮高度保持桌面端尺寸
  And 页面布局与桌面端基本一致
```

### Scenario 7: 移动端登录页面横屏模式

```gherkin
Given 用户在移动设备上访问登录页面
  And 设备处于横屏模式（宽度 > 高度）
When 页面加载完成
Then 登录表单宽度调整为屏幕宽度的 60%（最大 500px）
  And 表单垂直居中显示
  And 虚拟键盘弹出时，表单自动调整位置以保持可见
```

### Scenario 8: 移动端登录页面 Dark Mode 支持

```gherkin
Given 用户在移动设备上访问登录页面
  And 系统处于 Dark Mode
When 页面加载完成
Then 登录表单背景色适配 Dark Mode
  And 输入框背景和边框颜色在 Dark Mode 下清晰可见
  And 文字颜色在 Dark Mode 下清晰可读
  And 错误提示颜色在 Dark Mode 下清晰可辨
```

## 变更规格

### 登录表单移动端样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 表单宽度 | `w-[400px]` | `w-[90%] max-w-[400px]` |
| 表单内边距 | `p-8` | `p-6` |
| Logo 大小 | `h-12` | `h-10` |
| 标题字体 | `text-2xl` | `text-xl` |
| 输入框高度 | `h-10` | `h-11` (44px) |
| 输入框字体 | `text-sm` | `text-base` (16px) |
| Label 字体 | `text-sm` | `text-sm` (14px) |
| 按钮高度 | `h-10` | `h-11` (44px) |
| 按钮字体 | `text-sm` | `text-base` (16px) |

### 登录页面响应式断点

| 屏幕宽度 | 表单宽度 | 内边距 | 字体大小 |
|----------|----------|--------|----------|
| < 640px (移动端) | `w-[90%] max-w-[400px]` | `p-6` | 增大（16px 输入框） |
| 640px - 768px (大屏手机) | `w-[400px]` | `p-6` | 增大（16px 输入框） |
| 768px - 1024px (平板) | `w-[500px]` | `p-8` | 桌面端尺寸 |
| > 1024px (桌面端) | `w-[400px]` | `p-8` | 桌面端尺寸 |

### 移动端错误提示样式

| 属性 | 桌面端 | 移动端 |
|------|--------|--------|
| 字体大小 | `text-sm` | `text-sm` (14px) |
| 内边距 | `p-2` | `p-3` |
| 图标大小 | `w-4 h-4` | `w-5 h-5` |
| 边框圆角 | `rounded` | `rounded-md` |
